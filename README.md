# LineCoding
This SIMULINK model contains almost all of the famous line coding techniques.
It uses the Bernoulli Binary Generator to generate a binary string and generates the following line codes:
1.	Unipolar NRZ
2.	Unipolar RZ
3.	Polar NRZ
4.	Polar RZ
5.	Bipolar NRZ
6.	Bipolar RZ
7.	Manchester
